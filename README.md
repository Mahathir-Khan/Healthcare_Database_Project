# Healthcare_Database_Project
CSC 261 Database Project

## Run locally

Open `index.html` directly in a browser, or run a local server from this folder:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages

This project is set up for GitHub Pages from the repository root. In GitHub,
enable Pages with the source set to the `main` branch and `/ (root)`.
